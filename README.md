# MarketStacking
大学生创新创业训练项目代码
该项目是一个商业二分类预测问题。使用了两个著名公开数据集，一个是国外服装店数据集，另外一个是葡萄牙某银行数据集。数据集放在文件夹中，无需另外下载。

### 主要任务
完成了对于特定数据集一些经典模型的实现。例如LR、DT、KNN、XGBoost等。实验证明我们构建的stacking模型具有最好的效果。
